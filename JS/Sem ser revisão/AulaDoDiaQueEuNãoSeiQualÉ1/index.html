Condicional simples

Estuturas condicionais servem para alterar o fluxo de exucução do meu algoritimo. Apos um teste de condição, se a condição for verdadeira, o fluxo de execução do código interno à estrutural Condicional executa. Caso o contrario, o Codigo salta para o final da estrutura condicional.

if (teste de condição){
Comandos executados se a condição for verdadeira
}

Condicional Composta

if (teste de condição){
Comandos executados se a condição for verdadeira
}

else{
Comandos executados se a condição for verdadeira
}
